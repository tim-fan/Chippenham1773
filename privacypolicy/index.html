Privacy Policy

This privacy notice explains how your information is handled when you use this website, which provides a utility to show your location on a map.

  1. No Collection of Personal Information: Your location information is only used locally on your device, and is not sent to any external server. The site is hosted statically on Github pages, so is incapable of receiving or storing personal information, such as your location.
  2. Use of Cloudflare Analytics: Cloudflare Web Analytics is used to track website usage and gather non-personal information, such as the number of visitors, page views, and browser types. This is used to keep track of which maps are being used, and how frequently they are used. Quoting cloudflare.com/en-gb/web-analytics/:
    Cloudflare Web Analytics does not use any client-side state, such as cookies or localStorage, to collect usage metrics. We also don’t “fingerprint” individuals via their IP address, User Agent string, or any other data for the purpose of displaying analytics.
  3. Use of Cookies: This website does not use cookies.

If you have any questions or concerns about these privacy practices, please contact enquiries@onamap.me.
